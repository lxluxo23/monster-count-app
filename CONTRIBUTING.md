# Cómo contribuir

Gracias por tu interés en Monster Counter.

## Reportar bugs o sugerir mejoras

- **Bugs**: abre un issue con la plantilla [Bug](.github/ISSUE_TEMPLATE/bug_report.md).
- **Nuevas ideas**: usa la plantilla [Feature request](.github/ISSUE_TEMPLATE/feature_request.md).

## Pull requests

1. Haz fork del repo y crea una rama desde `master` (p. ej. `fix/typo-readme`).
2. Asegúrate de que el proyecto arranca: `npm install` y `npm start`.
3. Sigue el estilo existente: TypeScript, estructura en `src/`, tema en `src/theme`.
4. Abre un PR contra `master` y rellena la [plantilla de PR](.github/PULL_REQUEST_TEMPLATE.md).

## Seguridad

Para vulnerabilidades, no abras un issue público. Ver [SECURITY.md](SECURITY.md).
