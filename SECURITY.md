# Política de seguridad

## Versiones soportadas

Solo la última versión publicada recibe actualizaciones de seguridad.

| Versión | Soportada          |
|---------|---------------------|
| 1.0.x   | :white_check_mark:  |

## Cómo reportar una vulnerabilidad

Si encuentras una vulnerabilidad de seguridad, **no** abras un issue público.

- **Email**: indica en el repositorio o en tu perfil de GitHub un contacto para reportes de seguridad, o
- **GitHub Security Advisories**: en la pestaña *Security* del repositorio → *Report a vulnerability*.

Te responderemos lo antes posible y te mantendremos informado sobre el avance.

## Alcance

- La app guarda datos solo en el dispositivo (SQLite local).
- No se envían datos personales a servidores propios; cualquier servicio de terceros (p. ej. Expo/EAS) está sujeto a sus propias políticas.
